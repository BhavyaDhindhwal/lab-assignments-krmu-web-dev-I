# HTML Basics Project

This project is a simple one-page portfolio created using **basic HTML tags**.

This project is a simple Questions  created using **basic HTML tags**.  

It demonstrates how to structure a webpage with headings, paragraphs, lists, links, and images.


##  Features

- Basic HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)

- Headings (`<h1>` to `<h6>`)

- Paragraphs (`<p>`)

- Lists (`<ul>`, `<ol>`, `<li>`)

- Links (`<a>`)

- Images (`<img>`)

- Tables (`<table>`, `<tr>`, `<td>`)

- Forms (`<form>`, `<input>`, `<button>`)

##  Purpose

- To practice and understand fundamental HTML tags

- To build a simple portfolio layout

- To serve as a starting point for learning CSS and JavaScript later
